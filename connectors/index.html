Conectores omnicanal

Integraciones reales y funcionales. Las credenciales se guardan solo en tu navegador (localStorage) y las llamadas salen directo a cada API — sin servidores intermedios.

Por seguridad, tus credenciales viven solo en este dispositivo. Si usas un equipo compartido, pulsa "Borrar sesión y credenciales" al terminar. Usa siempre tokens con el mínimo alcance posible.

Asistente IA · Gemini

Sin conectarDocs

Pega tu API key de Google AI Studio y la IA queda lista para resumir correos, redactar y potenciar el dashboard. La clave se guarda solo en tu navegador.

GitHub

Sin conectarDocs

Repos, issues y PRs en vivo. Funciona con un usuario/organización público; añade un token (PAT) para repos privados y más límite.

Telegram

Sin conectarDocs

Bot de alertas omnicanal. Crea un bot con @BotFather, pega el token, y usa 'Leer updates' para descubrir el chat_id (escríbele algo al bot primero).

Gmail · Correo de la agencia

Sin conectarDocs

OAuth de Google (sin secreto) y lectura real de tu bandeja. El dashboard usará estos correos en vivo cuando esté conectado.

Crea un Client ID de tipo "Web" en Google Cloud Console y añade este origen a "Authorized JavaScript origins".

Google Drive

Sin conectarDocs

Explora archivos y carpetas reales (incluidas las compartidas y unidades compartidas). Navega entre carpetas y busca por nombre.

Configura primero el Google OAuth Client ID en la tarjeta de Gmail (es el mismo).

Google Calendar

Sin conectarDocs

Configura primero el Google OAuth Client ID en la tarjeta de Gmail (es el mismo).
